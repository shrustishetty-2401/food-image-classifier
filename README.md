# 🍔🍕 Food Image Classifier (CNN)

A deep learning project that classifies food images into categories like pizza and burger using a Convolutional Neural Network (CNN).

## 🚀 Tech Stack
- Python
- TensorFlow / Keras
- Jupyter Notebook
- CNN (Convolutional Neural Network)

## 📊 What it does
- Loads food images from dataset
- Trains a CNN model
- Classifies images into food categories

## 📁 Project Structure
- dataset/
- foodenv/
- notebook.ipynb

## 🧠 Model
- Convolutional layers
- MaxPooling layers
- Dense layers
- Softmax output

## 📌 Status
Project in development (adding more classes + improving accuracy)
